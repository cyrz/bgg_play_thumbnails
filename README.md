<h1>BGG Weekly Game Play Tool</h1>

<h2>This is really just a pet project. It's designed to work exactly for my need, and isn't meant to be extensible.</h2>

This will get me a week's worth of thumbnails for games that I played over a given week. This is used to add a little flair to the weekly [link=https://boardgamegeek.com/thread/1695590/what-did-you-play-week-wdyptw-subscription-thread]What Did You Play Geeklist[/link].

Based on the output of http://www.robandkriskris.com/gcl/
